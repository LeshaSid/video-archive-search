# Video Archive Search

Поиск по локальному видеоархиву через автоматическую транскрибацию речи и векторную базу данных.  
Вы указываете папку с видеофайлами — программа преобразует речь в текст, индексирует фрагменты по смыслу и позволяет мгновенно находить нужные моменты по текстовому запросу, запуская плеер с точной секунды.

---

## Возможности

- **Автоматическое распознавание речи (STT):** Локальная транскрибация аудиодорожек на русском языке с помощью высокоэффективной модели `faster-whisper` (версия `small` с квантованием `int8` для экономии процессора).
- **Семантический (векторный) поиск:** Использование эмбеддингов `intfloat/multilingual-e5-small` для поиска по смыслу и контексту, а не только по ключевым словам. Поисковый запрос может быть синонимичным или описывать суть своими словами.
- **Интегрированный видеоплеер:** Быстрый переход и воспроизведение найденного фрагмента прямо в интерфейсе приложения с конкретной секунды.
- **Умное кэширование и дедупликация:** При повторном сканировании папки система пропускает уже проиндексированные видеофайлы, сохраняя историю путей.
- **100% Локально и безопасно:** Все вычисления, модели и базы данных разворачиваются на вашей машине без отправки данных в облако или на сторонние API.

---

## Архитектура проекта

Проект имеет модульную и легковесную структуру:
- `main.py` — веб-интерфейс приложения на фреймворке **Streamlit**, логика сканирования директорий, прогресс-бары и кастомный плеер.
- `database.py` — интеграция с локальной векторной базой данных **Qdrant** (хранится в `.qdrant_db`) и генерация эмбеддингов через **SentenceTransformers**.
- `transcriber.py` — извлечение и обработка аудио через **static-ffmpeg**, фильтрация тишины и голосовой активности (VAD) и разбивка текста на смысловые чанки для эффективного поиска.

## Системные требования

- **ОС:** Windows 10/11, macOS или Linux (64-бит).
- **Python:** Версия `3.12`.
- **Свободное место:** Около 2 ГБ для локального хранения весов ИИ-моделей и базы данных Qdrant.
- **Железо:** Рекомендуется минимум 8 ГБ ОЗУ. Текущая конфигурация оптимизирована для работы на CPU.

## Установка и запуск

### Способ 1: С помощью менеджера пакетов `uv` (Рекомендуемый)

Проект использует современный и быстрый менеджер пакетов `uv` от Astral.

1. Клонируйте репозиторий:
```bash
git clone [https://github.com/LeshaSid/video-archive-search](https://github.com/LeshaSid/video-archive-search)
cd video-archive-search

```
2. Синхронизируйте окружение и установите зависимости (зависимости зафиксированы в `uv.lock`):
```bash
uv sync

```


3. Запустите Streamlit-приложение:
```bash
uv run streamlit run main.py

```



> **Важно:** При первом запуске или первом сканировании система автоматически скачает модели (Whisper small и E5-small transformer). Это может занять несколько минут в зависимости от скорости вашего интернета.

### Способ 2: Быстрый запуск на Windows (через `run.bat`/`run.exe.`)

Для пользователей Windows предусмотрен автоматический скрипт запуска `run.bat`:

1. Скачайте/склонируйте проект в папку.
2. Дважды кликните по файлу `run.bat`(либо по `run.exe`, если скачиваете релиз).
3. Скрипт самостоятельно:
* Проверит наличие `uv` в системе (и установит его в профиль пользователя, если он отсутствует).
* Выполнит синхронизацию окружения `uv sync` в локальный `.venv`.
* Автоматически откроет окно браузера с адресом `http://127.0.0.1:8501` и запустит сервер Streamlit в фоновом автономном режиме.
> Примечание: `run.exe` должен находится в корне проекта.

## Как пользоваться

1. **Индексация архива:**
* В левой боковой панели введите абсолютный путь к папке с вашими видеофайлами (поддерживаются форматы: `.mp4`, `.mkv`, `.mov`, `.avi`).
* Нажмите кнопку **«Начать сканирование»**. В интерфейсе отобразится прогресс-бар и текущий обрабатываемый файл.


2. **Поиск моментов:**
* После завершения индексации в центральное поле ввода введите интересующий вас запрос (например: *«Коля ест сахар»* или *«обсуждение архитектуры базы данных»*).
* Система выдаст топ-5 наиболее релевантных текстовых фрагментов.


3. **Воспроизведение:**
* Кликните на карточку-кнопку с найденным текстом.
* Справа откроется встроенный видеоплеер, который автоматически перемотает видео на таймкод начала этой фразы.

## Стек технологий

* **UI-фреймворк:** [Streamlit](https://streamlit.io/)
* **Распознавание речи:** [Faster-Whisper](https://github.com/SYSTRAN/faster-whisper) (движок CTranslate2)
* **Векторная база данных:** [Qdrant](https://qdrant.tech/) (встраиваемый локальный клиент)
* **Текстовые эмбеддинги:** `intfloat/multilingual-e5-small` via [Sentence-Transformers](https://sbert.net/)
* **Менеджер пакетов:** [uv](https://github.com/astral-sh/uv)
* **Декодирование медиа:** `static-ffmpeg`
