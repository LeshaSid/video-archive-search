# Video Archive Search
Поиск по видеофайлам через транскрибацию речи и векторную базу данных.  
Вы указываете папку с видео (MP4, MKV, MOV, AVI), программа расшифровывает речь (русский язык), индексирует фрагменты и позволяет находить нужные моменты по текстовому запросу.

## Возможности

- Автоматическая транскрибация видео (faster-whisper)
- Векторный поиск по смыслу, а не только по точному совпадению
- Просмотр найденного фрагмента прямо в интерфейсе с переходом на нужную секунду
- Работает полностью локально, без облачных сервисов

## Установка и запуск

### Через uv (рекомендуемый способ)

```bash
git clone https://github.com/LeshaSid/video-archive-search
cd video-archive-search
uv sync
uv run streamlit run main.py
```

При первом запуске скачаются модели (Whisper small, sentence‑transformer). Это займёт некоторое время.

### На Windows (альтернатива)

Просто запустите `run.bat` — скрипт сам создаст портативное окружение с Python, установит зависимости и откроет приложение в браузере.

## Использование

1. В боковой панели укажите путь к папке с видеофайлами.
2. Нажмите «Начать сканирование» — начнётся транскрибация и индексация.
3. После завершения введите поисковый запрос в главное поле.
4. Нажмите на любой результат поиска — видео откроется в плеере с нужного момента.

## Требования

- Python 3.12
- Около 2 ГБ свободного места под модели и базу данных
- 64‑битная операционная система (для работы моделей ИИ)
